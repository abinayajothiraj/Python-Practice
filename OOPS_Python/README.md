# Python OOP Basics

This folder contains beginner-friendly Object-Oriented Programming (OOP) examples in Python.

The examples are written with detailed comments and explanations to understand how OOP concepts work internally.

---

# Concepts Covered

- Classes and Objects
- Constructors using `__init__`
- The `self` keyword
- Instance Variables
- Class Variables
- Methods inside classes
- Multiple methods in a class
- Default parameters
- Object string representation using `__str__`
- Adding and removing data from objects
- Loops with OOP
- Deleting methods using `del`
- Difference between class variables and instance variables

---

# Files Included

## 1. car_class.py

Covers:
- Creating classes and objects
- Using `__init__`
- Instance variables
- Class variables
- Object methods

Examples:
- Car brand storage
- Shared class variables (`wheels`)
- Overriding class variables with instance variables

## 2. person_class.py

Covers:
- Constructors with default parameters
- Methods inside classes
- Creating multiple objects

Examples:
- Person class with city and country
- Greeting method
- Dog class with bark method

## 3. class_methodss.py

Covers:
- `__str__()` method
- Multiple methods inside a class

Covers:
- Deleting methods using `del`
